# pyprop

A simple numerical propagator implemented in python.
This was done purely for fun/educational purposes.
It is neither very efficient nor very accurate.

## Force modelling
The propagator can consider
- Central Body Gravity of Earth
- Zonal Harmonics of the Earth gravity field up to J4
- Atmoshperic Drag using and exponential Model
- 3rd Body gravity of Sun and Moon
- Solar radiation pressure
